# Story 3.3: Perfil Completo e Metricas Agregadas do Influencer

| Campo | Valor |
|-------|-------|
| **Epic** | Epic 3: Gestao de Influencers |
| **Status** | Ready |

## User Story

**Como** gestor de marca,
**Quero** ver um perfil completo do influencer com todas as metricas e campanhas,
**Para que** eu tenha uma visao 360 de cada influencer.

## Acceptance Criteria

- [x] 1. Pagina de perfil do influencer com: dados pessoais, cupom, links de redes sociais
- [x] 2. Secao de metricas agregadas: total de vendas geradas, receita total, ROI medio, nr de campanhas (placeholder ate Epic 6)
- [x] 3. Lista de campanhas participadas com: nome, periodo, vendas, receita, ROI individual (placeholder ate Epic 6)
- [x] 4. Metricas calculadas com dados mock/placeholder (implementacao real no Epic 6)
- [x] 5. Layout responsivo com cards de KPI no topo + tabelas detalhadas abaixo

## Technical Notes

- Pagina dinamica: `src/app/(dashboard)/influencers/[id]/page.tsx`
- KPI cards com Skeleton loader (dados reais vem no Epic 6)
- Campanhas placeholder: tabela vazia com mensagem "Nenhuma campanha vinculada"
- Metricas agregadas: usar views `v_influencer_metrics` quando implementadas (Epic 6)
- Integrar historico de crescimento da Story 3.2

## Dependencies

- Story 3.1 (CRUD de influencers)
