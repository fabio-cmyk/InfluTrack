# Story 3.2: Historico de Crescimento e Metricas

| Campo | Valor |
|-------|-------|
| **Epic** | Epic 3: Gestao de Influencers |
| **Status** | Ready |
| **Prioridade** | P1 |

## User Story

**Como** gestor de marca,
**Quero** registrar e visualizar o historico de crescimento de cada influencer,
**Para que** eu acompanhe a evolucao de seguidores e engajamento ao longo do tempo.

## Acceptance Criteria

- [x] 1. No perfil do influencer, secao "Historico de Crescimento" com grafico de linha
- [x] 2. Tabela `influencer_growth_history` com: influencer_id, date, platform, followers, engagement_rate, posts_count (migration 00003)
- [x] 3. Formulario para adicionar entrada manual de crescimento (data, plataforma, seguidores, engajamento)
- [x] 4. Grafico de linha mostrando evolucao de seguidores por plataforma ao longo do tempo
- [x] 5. Card com metricas atuais: seguidores totais por plataforma, taxa de engajamento mais recente
- [x] 6. Ordenacao cronologica com data mais recente primeiro na tabela

## Technical Notes

- Tabela `influencer_growth_history` na migration `00003_create_influencers.sql`
- Campos: influencer_id, tenant_id, record_date (DATE), platform (instagram/tiktok/youtube), followers, engagement_rate (DECIMAL 5,2), posts_count
- UNIQUE constraint em (influencer_id, record_date, platform)
- Usar Recharts para graficos de linha (ja instalado)
- RLS: influencers.view para SELECT, influencers.edit para INSERT
