## User Story

**Como** desenvolvedor,
**Quero** ter o projeto Next.js configurado com Supabase, linting, testes e estrutura de monorepo,
**Para que** a equipe tenha uma base sólida para desenvolver todos os módulos.

## Acceptance Criteria

- [x] 1. Projeto Next.js 14+ (App Router) inicializado com TypeScript strict mode
- [x] 2. Tailwind CSS e shadcn/ui configurados e funcionais
- [x] 3. Supabase client configurado com variáveis de ambiente (.env.local)
- [x] 4. Estrutura de monorepo criada: `packages/web`, `packages/shared`, `supabase/`
- [x] 5. Vitest configurado com pelo menos 1 teste passando
- [x] 6. ESLint + Prettier configurados com regras do projeto
- [x] 7. Layout base com sidebar de navegação (links placeholder para todos os módulos)
- [x] 8. Página inicial renderizando com layout correto
- [x] 9. README com instruções de setup local

## Technical Notes

- Stack: Next.js 14+ (App Router), Tailwind CSS, shadcn/ui, Supabase, Vitest
- Monorepo com Turborepo
- Design tokens de `docs/design/influtrack-frontend-spec.md`
- Layout da sidebar conforme seção 3.1 do frontend spec
